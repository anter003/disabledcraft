# 비행선
> 비행선들은 이착륙, 충돌 시마다 내구도가 닳습니다. 우클릭으로 수리하세요.

> 비행을 위해서는 연료(석탄, 숯 등)이 필요합니다. `E`를 눌러 비행선 메뉴에 접근하세요. 

> 일부 기체는 폭죽 로켓을 이용해 부스트가 가능합니다.

![](https://cdn.modrinth.com/data/cached_images/d1e925cdcb7d4212c3ec688af6ab9017f8a2fde3.jpeg)

# 무기
> 맨손 + `우클릭`으로 무기를 작동시킬 수 있습니다. 

* 로터리 캐논(Rotary Cannon) : `화약` 필요
* 무거운 석궁(Heavy Crossbow) : `화살` 필요
* 폭탄 만(Bomb Bay) : `TNT` 필요, 실제로 폭발하진 않고 데미지만
* 망원경(Telescope) 

# 업그레이드

* 향상된 프로펠러(Enhanced Propeller) : 공기 마찰 -75%
* 산업용 기어(Industrial Gears) : 연료 요구량 -20%
* 견고한 파이프(Sturdy Pipes) : 엔진 출력 +10%
* 선체 강화(Hull Reinforcement) : 내구성 +100%
* 향상된 랜딩 기어(Improved Landing Gear) : 이륙 속도 +50%
* 자이로스코프(Gyroscope) : 안정화 +3%, 바람 효과 -100%


* 에코 엔진(Eco Engine) : 엔진 출력 -20%, 연료 요구량 -75%
* 네더 엔진(Nether Engine) : 엔진 출력 +40%, 연료 요구량 +30%
* 강철 보일러(Steel Boiler) : 엔진 출력 +25%, 연료 요구량 +50%

# 조작법
> :warning: 비행 중 W를 과하게 누르면 선체가 앞으로 추락할 수 있습니다.

> `space`로 출력을 높이고 `s`를 눌러 이륙하세요.

* `space` : 엔진 출력 증가
* `shift` : 엔진 출력 감소
* `W` : 전진/위로
* `S` : 후진/아래로
* `A/D` : 조종
* `E` : 비행선 인벤토리 열기
* `R` : 내리기
* `우클릭` : 무기 사용
* `B` : 폭죽 로켓 부스트
